Boolean(布林值)

布林值其實就是電腦科學中的邏輯判斷資料型態，其型態共有兩種True 與 False ，分別代表正確與錯誤

Python中 0 或 空(empty) 代表 False ， 1 或 其餘元素 代表False，其餘的皆代表True (通常 0 或 1 比較常用)

Python中，boolean縮寫為：bool

statement = "今天請大家吃王品"
my_age = 18

print(bool(statement)) #True
print(bool(my_age)) #True
print(bool(1)) #True

print(bool(False)) #False
print(bool(None)) #False
print(bool(0)) #False

比較運算符號

電腦科學中通常有6種比較關係： 比較運算符號的兩側，可以放常數、變數、運算式 比較的結果型態為boolean

小於 大於 小於等於 大於等於 等於 不等於
< > <= >= == != 
num1 = 100
num2 = -10

print(bool(num1 <= num2)) #False
print(bool(num1 != num2 + 2)) #True

比較運算出來的結果一定為boolean值，因此也可以省略 bool() 型態轉換

num1 = 100
num2 = -10

print(num1 <= num2) #False
print(num1 != num2 + 2) #True

if 條件選擇

條件選擇，是藉由判斷條件，以選擇接下來要不要執行某一段程式

s = "cool"

if s == "cool":   #字串也適用
    print("Yes!") #條件成立所以能執行這列程式

if 1:
    print("Yes!") #條件成立所以能執行這列程式

注意! Python中縮排是非常重要的，代表縮排的那幾列程式碼都屬於if判斷式下的程式碼，若判斷結果為False，則屬於其下方的程式碼都不會被執行

if 0:
    print("NO!") #條件不成立所以不能執行這列程式

else

else 出現的時機就是不符合某個判斷條件時才要執行的程式，所以它一定要接在某個if後面

請記得if的區塊和else的區塊之間絕不能放任何程式(除了無意義的空行或空白以外)

if 2 > 3:
    print("正確")
    print("超過100分")
else:
    print("錯誤") #if條件不成立會執行這列，若成立則不執行

且 或 否
and or not

False T F

多個相互獨立的條件判斷

各條件判斷間沒有關聯性

【例題】

爸爸為了想讓我數學科目分數更高些，於是提議：
如果我第一次段考超過70分就頒發1000元獎勵金
如果我第二次段考超過80分就頒發2000元獎勵金
如果我第三次段考超過90分就頒發3000元獎勵金
分別輸入3次段考成績(g1, g2, g3)
輸出最終我可以拿到多少獎勵金？ 
g1, g2, g3 = input("請輸入三次段考成績：").split()

total = 0 #先創造一個變數用來存放獎金值

if int(g1) >= 70:
    total += 1000
if int(g2) >= 80:
    total += 2000
if int(g3) >= 90:
    total += 3000

print(total)

2020武漢肺炎全球大流行，政府決定對有出國史的人進行強制居家隔離，並對有出國史且有感冒者進行醫院隔離

trip = input("請問您14天內是否有出國過?(Y/N)：")
if trip == "Y":
    illness = input("請問目前身體是否有不適?(Y/N)：")
    if illness == "Y":
        print("醫院隔離")
    else:
        print("居家隔離")
else:
    print("不用隔離")

由於地球公轉一周大約是365.242199日，所以如果以一年365日來記算，每四年就會多出0.968796天，於是有人提出每四年要多出一天，這就是閏年。但是因為每四年多出來的並非完整的一天，所以累積到100年的時候，就不須要再多這一天了。而同樣的道理，到了400年，又會多出一天來。

目前閏年的規則如下(西元紀年)：
1. 如果這一年不是4的倍數，則它是平年。
2. 如果這一年是4的倍數，但不是100的倍數，則它是閏年。
3. 如果這一年是100的倍數，但不是400的倍數，則它是平年。
4. 如果這一年是400的倍數，則它是閏年。

輸入說明：
輸入一個正整數 N，代表這是西元 N 年。
輸出說明：
若這一年是閏年，則輸出 YES，否則輸出 NO。

提示：

i = input("請輸入西元：")
i = int(i)
if i % 4 == 0:
    # 請填入程式碼
else:
    print("NO")

上面這範例利用許多 if 達成題目的要求，但其實這有點拖慢了程式執行的速度，畢竟所有星座都是互斥的，也就是不可能有人同時具有兩個星座(如果我是魔羯座，就不可能也是射手座或金牛座)

因此

其實我們可以有更好的寫法提升程式執行的效率：elif

elif 只要前面任一列程式條件成立，後面的就不會再判斷了，因此能省下許多時間

m, d = input("請輸入您的出生年月日：").split()
m = int(m)

if (m == 1 and d < 20) or (m == 12 and d > 21):
