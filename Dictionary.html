字典

Dictionary, Tuples, Sets

tags: Python

什麼是Dictionary?

直接英翻中的話就稱為：字典
因此，其實Dictionary就像大家小時候查字典的方式相同
我們會利用部首 or 筆畫數 or 注音符號索引，找到我們想找的內容在哪一頁
在Python中，我們也是利用索引值(Key)去找相對應的內容(Value)唷!!

字典有甚麼優勢

有沒有想過，其實儲存大量資料也可以利用陣列就好
為什麼需要有「字典」這種資料型態呢?

創造一個字典

字典也是必須先為它取個名稱

方法一

{}中放上一些<Key>：<Value>
每一對<Key>：<Value>利用逗號分隔

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
方法二

直接利用dict()將它變為dict的型態

thisdict = dict(brand="Ford", model="Mustang", year=1964)
【試試看】 
thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

print(thisdict) #試著印出來看看它的模樣

利用key值找到相對應的value

利用<dict_name>[<key>] 或 <dict_name>.get(<key>)

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

print(thisdict["model"]) #Mustang
print(thisdict.get("year")) #1964

改變某key值所對應的value

直接利用<dict_name>[<key>] = <new_value> 即可

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

thisdict[year] = 2020

print(thisdict.get("year")) #2020

新增新的key值和value

直接利用<dict_name>[<new_key>] = <new_value> 即可

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

thisdict["pay"] = 50000

print(thisdict) #{'brand': 'Ford', 'model': 'Mustang', 'year': 1964, 'pay': 50000}

刪除特定key值和所對應的value

直接利用del <thisdict>[<key>] 或 <thisdict>.pop(<key>)

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964,
  "pay": 50000
}

del (thisdict["model"])

print(thisdict) #{'brand': 'Ford', 'year': 1964, 'pay': 50000}

若利用<thisdict>.popitem()會直接去除掉字典中最新(最後加進去)的那一筆

列出所有key值 或 value值

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964,
  "pay": 50000
}

#列出所有key值
for x in thisdict:
  print(x) 

#列出所有value值
for x in thisdict:
  print(thisdict[x]) 

#列出所有value值
for x in thisdict.values():
  print(x)

#列出所有 key 及其相對應的 value
for x, y in thisdict.items():
  print(x, y)

確認某key值是否存在字典內

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

if "model" in thisdict:
  print("Yes")

確認字典的總資料筆數

利用len(<dict_name>)

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

print(len(thisdict)) #3

複製整個字典

利用<dict_name>.copy()

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = thisdict.copy()
print(mydict) #{'brand': 'Ford', 'model': 'Mustang', 'year': 1964}

清空字典中的資料

利用<dict_name>.clear()

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = thisdict.clear()
print(mydict) #None
