tags: Python

迴圈的用途

  • 重複地做一樣的事情
  • 不知道要做幾次才結束，但用某個條件判斷決定是否結束

想想看，日常生活中有那些不斷重複做的事情呢?

while迴圈

while迴圈的基本架構

i = 0 # 從0開始
while i < 10:  # i<10 就是每一次進入迴圈與否的條件判斷
    print(i)
    i += 1

### `while`迴圈結合`if`判斷式 在前面的章節中我們常常會執行並輸入值給程式
但有沒有覺得每次要輸入都得重按一次執行鈕非常麻煩
學會利用迴圈，我們就可以不必每次輸入都得重新執行囉! 【例題】 
while 1:
    num = int(input("請輸入一個分數:"))
    if num >= 60:
        print("及格")
    elif 40 <= num <60:

    else:
        print("死當")

while迴圈搭配break或continue

有時候在迴圈中的某個情況發生時，我們希望終止迴圈或跳過此圈繼續執行下一回圈，這時我們就會使用到break或continue

  • break：當電腦執行到break時會立即終止該層迴圈
  • continue：當電腦執行到continue時會立即跳過該輪迴圈，直接進行下一輪

【例題1】 寫一個程式，印出1-15但不印出13

i = 1

while i <= 15:
    if i == 13:
        i+=1 #想想看為什麼要有這一列，如果這一列放在continue後會如何?
        continue #跳過i==13這論迴圈
    print(i)
    i+=1

【例題2】 寫一個程式，能夠不斷輸入成績判斷是否及格，但若輸入值為-1時，結束執行該程式

while 1:
    num = int(input("請輸入一個分數:"))
    if num == -1:
        break
    if num >= 60:
        print("及格")
    elif 40 <= num < 60:
        print("可補考")
    else:
        print("死當")

for迴圈

for迴圈則跟while迴圈非常相似，最主要區別是在while迴圈通常會用在不知道要執行多少次的情況下，而for迴圈通常用在已經知道要執行多少次的情況

例如：
印出100個"Hello"
算出某個數字的某次方
算出某個數字的階乘…等

上述情況就比較適合使用for迴圈
