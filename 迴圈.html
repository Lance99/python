迴圈

tags: Python

迴圈的用途

  • 重複地做一樣的事情
  • 用某個條件判斷決定是否結束不斷重複的輪迴

想想看，日常生活中有那些不斷重複做的事情呢?

while迴圈

while迴圈的基本架構

通常迴圈都會需要有個變數用來計算目前執行了幾圈
此變數名稱可以用稍微簡易一些的。例如：ｉ 
i = 0 #用於計算執行了幾圈從，0開始
while i < 10:  # i<10 就是每一次進入迴圈與否的條件判斷
    print(i)
    i += 1 #每圈結束後讓計算變數值改變

注意! 縮排也別忘了唷!

while迴圈搭配break或continue

有時候在迴圈中的某個情況發生時，我們希望終止迴圈或跳過此圈繼續執行下一回圈，這時我們就會使用到 break 或 continue

  • break：當電腦執行到break時會立即終止該層迴圈
  • continue：當電腦執行到continue時會立即跳過該輪迴圈，直接進行下一輪
【例題1】
寫一個程式，印出1~15但不印出13 
i = 1

while i <= 15:
    if i == 13:
        i+=1 #想想看為什麼要有這一列，如果這一列放在continue後會如何?
        continue #跳過i==13這論迴圈
    print(i)
    i+=1
【例題2】
寫一個程式，能夠不斷輸入成績判斷是否及格，但若輸入值為-1時，結束執行該程式 
while 1:
    num = int(input("請輸入一個分數:"))
    if num == -1:  # 假設這個if條件式放在最後會怎樣呢?
        break
    if num >= 60:
        print("及格")
    elif 40 <= num < 60:
        print("可補考")
    else:
        print("死當")

for迴圈

for迴圈則跟while迴圈非常相似，最主要區別是在while迴圈通常會用在不知道要執行多少次的情況下，而for迴圈通常用在已經知道要執行多少次的情況

例如：

1. 印出100個Hello
2. 算出某個數字的某次方
3. 算出某個數字的階乘…等

上述情況就比較適合使用for迴圈

for迴圈的基本架構

for迴圈中也必須有一個變數用來計算目前執行了幾圈
計算圈數的變數寫在for迴圈結構中
每輪迴圈完畢後，計算圈數的變數會自動變化(沒特別說明就是+1) 
for i in range(6): # 從0-5，注意不包含6；i 每輪都+1
    print(i)

for i in range(2, 6):  # 從2-5，注意不包含6；i 每輪都+1
    print(i)

for x in range(21, 3, -3): # x 每輪 -3
    print(x)

【例題】
將b a n a n a字串改為B a N a N a 
ss = "banana"
result = "" #先創造一個最後要輸出的字串
for i in range(0, len(ss)):
    # 如果遇到 b
    if ss[i] == "b":
        # 將result字串中相對應的格子放入 B
        result = result + "B"
    # 如果遇到 n
    elif ss[i] == "n":
        # 將result字串中相對應的格子放入 N
        result = result + "N"
    # 如果遇到其它字元
    else:
        # 直接放入result字串中相對應的格子
        result = result + ss[i]
print(result)
