輸入與變數

輸入與變數

tags: Python

什麼是變數

相信之前如果有學過一點點程式設計的大家都已經對「變數」這名稱不陌生了，我們可以簡單地把變數想像成為一個盒子，這個盒子可以一次裝一筆資料進去，但當我們再放新的一筆資料進盒子中時，前一筆資料就會被迫消失囉!

為何需要變數

電腦畢竟是人類所開發出的產品，因此思維模式肯定與人類相去不遠，人類不論從眼、耳、鼻、舌、手等感知器官所偵測到的訊息都必須存放在腦中，因此，電腦也同樣需要一個空間儲存所得到的訊息，那就利用變數吧!

Python變數命名規則

任何變數都需要被取個名字唷!!!

  1. 變數名稱的開頭必須是英文字母或底線

    _number tem2p Act_3 這些都是允許的變數名稱

  2. 不得由數字開頭當變數名稱

    7zz 2u 這種都不允許

  3. 變數名稱只能含有(A~z, 0~9, 和 _ )

    Act-3 也是不被允許的名稱唷

  4. 變數名稱的英文字母大小寫是有區別的

    Aoe aOE aoe 各是不同的變數

  5. 關鍵字不得當變數名稱

指定變數的值

box = 100 # box變數放入100
box = 300 # box變數放入300
s_s = "I am pretty" # s_s變數放入I am pretty
num, ss, ff = 300, "I come, I see, I conquer.", 3.1416 # 一次放入多項

注意! 資料放入至變數的盒子中在程式的世界稱為：指定(assign)

注意! Python這程式語言的變數不需要事先宣告，這與其他程式語言稍有不同唷! 例如：num = 100 變數 num 型態就是整數，ss = "Handsome" 變數 ss 型態就是字串

資料型態

Python 的內建型態主要分為以下三種：

  1. 數值型態：int, float, bool
  2. 字串型態：str, chr
  3. 容器型態：list, dict, tuple

    容器型態未來才會用到先不說明

資料型態的轉換：

Python資料型態轉換非常方便 int(a)：將變數a轉換成為整數

float(b)：將變數b轉換成為浮點數

str(c)：將變數c轉換成為字串

但是，不能轉的型態就是無法轉，例如：

ss = "hello" 
int(ss)

確認資料型態：

type(a)：變數a的資料型態

type(3.14)：3.14的資料型態

type("It's always a good time")：It's always a good time的資料型態

輸出變數

在 Python 中，印出資料在螢幕上是用 print，之前我們印字串需要在小括號中加上雙引號，印出變數內的資料時我們不需要加上雙引號唷!

例如：print(num) 即 印出變數num內的值

num = 1000 #將num變數值指定為1000
print(num) #1000

操作一下變數的四則運算

n1 = 1000
n2 = 33 
print(n1 % n2) #嘗試把'%'換成其它四則運算符號

什麼是輸入

如果程式只能夠輸出東西到螢幕上卻無法與人互動，這樣是不是顯得太無趣了呢？因此，本小節我將教大家如何讓電腦讀取我們所輸入給電腦的資訊唷!

從電腦中讀取到的資料，我們必須用一個變數將它們存起來

在Python中，我們利用input()作為讀取鍵盤輸入資料的工具 例如：ff = input() 變數ff會存放我們鍵盤所輸入的值

執行這列程式時，電腦會等待我們從鍵盤上輸入值給ff變數

ff = input() #假設鍵盤輸入3.1416
print(ff) #3.1416

input提示字串：

在input後的小括號中放入一列字串可以提示使用者如何輸入正確數值給此程式

ff = input("請輸入一個小數") #假設鍵盤輸入3.1416
print(ff) #3.1416

利用+ 來串接兩個字串：

ff = input("請輸入一個小數") #3.1416
print("你輸入的值為：" + ff) #你輸入的值為：3.1416

一次輸入多個值給不同變數：

a, b, c = input().split()

注意! input()從鍵盤讀取到的值資料型態都會預設為字串 所以上方a b c變數的型態均為str

一次輸入多個值給不同變數並將型態由字串轉為整數：

a, b, c = map(int, input().split()) # 2 -3 150
print(type(a)) # <class 'int'>
