什麼是變數

相信之前如果有學過一點點程式設計的大家都已經對"變數"這名稱不陌生了，我們可以簡單地把變數想像成為一個盒子，這個盒子可以一次裝一筆資料進去，但當我們再丟新的一筆資料進盒子中時，前一筆資料就會被迫消失囉!

box = 100 #box變數放入100
box = 300 #box變數放入300

【想想看】

  2. 在程式碼中，要指定值給變數是利用 = 的符號，那麼等於要用什麼樣的符號呢?

注意! Python這程式語言的變數不需要事先宣告，這與其他程式語言稍有不同唷! 例如：num = 100 變數 num 型態就是整數，ss = "Handsome" 變數 ss 型態就是字串

刪除變數：

del a (刪除變數 a)

num = 300 ss = "I am pretty" num, ss, ff = 300, "I am pretty", 3.1416 一次指定多項

資料型態

Python 的內建型態主要分為以下三種：

  1. 數值型態：int, float, bool
  2. 字串型態：str, chr

    字串(str)：一串字元(chr)，例如：WOW 今天是情人節耶!!! 字元(chr)：人類語文的最基本單位，例如：中文字、英文字母、阿拉伯數字、標點符號等。在Python中用單引號刮住，例如：'A'或'%'

  3. 容器型態：list, dict, tuple

    容器型態未來才會用到先不說明

資料型態的轉換：

Python資料型態轉換非常方便 int(a)：將變數a轉換成為整數 float(b)：將變數b轉換成為浮點數 str(c)：將變數c轉換成為字串

確認資料型態：

type(a)：變數a的資料型態 type(3.14)：3.14的資料型態 type("It's always a good time")：It's always a good time的資料型態

什麼是輸入

如果程式只能夠輸出東西到螢幕上卻無法與人互動，這樣是不是顯得太無趣了呢？因此，本小節我將教大家如何讓電腦讀取我們所輸入給電腦的資訊唷!

從電腦中讀取到的資料，我們必須用一個變數將它們存起來

在Python中，我們利用input()作為讀取鍵盤輸入資料的工具 例如：ff = input() 變數ff會存放我們鍵盤所輸入的值

執行這列程式時，電腦會等待我們從鍵盤上輸入值給ff變數

ff = input() #假設鍵盤輸入3.1416
print(ff) #3.1416

input提示字串：

在input後的小括號中放入一列字串可以提示使用者如何輸入正確數值給此程式

ff = input("請輸入一個小數") #假設鍵盤輸入3.1416
print(ff) #3.1416

利用 '+' 來串接兩個字串：

ff = input("請輸入一個小數") #3.1416
print("你輸入的值為：" + ff) #你輸入的值為：3.1416

【想想看】 1+1 => 2 "Hand"+"Some" => HandSome "5"+"6" => ???

一次輸入多個值給不同變數：

a, b, c = input().split()

注意! input()從鍵盤讀取到的值資料型態都會預設為字串 所以上方a b c變數的型態均為str
